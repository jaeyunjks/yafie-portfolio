# Content Plan - Yafie Portfolio

## Page-by-Page Checklist
- Home: Headline, value proposition, short intro, featured projects, CTA
- About: Personal profile, tech strengths, capability cards, exploration section
- Work: Experience entries grouped by category with achievements
- Projects: Featured and supporting projects, stack tags, action links
- Contact: Availability, preferred contact methods, collaboration CTA

## Placeholder Sections to Replace
- Personal bio and headline wording
- Actual project links (GitHub and live demo)
- Real contact details (email/phone/location)
- Real metrics and outcomes in work/project achievements
